🚗 BMW Cars Market Analysis Using Python

I recently completed an Exploratory Data Analysis (EDA) project using a BMW cars market dataset to strengthen my practical data analysis skills with Python. The dataset contains 10,000 records and includes several important features such as model, production year, engine size, fuel type, mileage, CO₂ emissions, price, and the country where the car was sold.

The main objective of this project was to explore the dataset, clean the data, and extract meaningful insights related to BMW car pricing, model popularity, and market distribution across different regions. Through this project, I aimed to better understand how different variables influence vehicle prices and market trends.

🔎 Project Workflow

• Performed initial data inspection to understand the structure, variables, and data types in the dataset
• Identified and handled missing values in both numerical and categorical columns using appropriate data cleaning techniques
• Conducted exploratory data analysis to examine distributions, detect patterns, and explore relationships between variables
• Used grouping and aggregation techniques to analyze the average price of different BMW models
• Investigated the distribution of fuel types and examined how BMW vehicles are distributed across different countries

📊 Visualizations Created

To better understand the dataset and communicate insights effectively, several visualizations were developed:

• Distribution of BMW car prices to understand price variation in the market
• Model-wise distribution to identify which BMW models appear most frequently
• Fuel type distribution showing the proportion of petrol, diesel, hybrid, and electric vehicles
• Scatter plot showing the relationship between mileage and car price
• Comparison of average prices across BMW models to highlight premium and entry-level segments

📈 Key Insights

• Premium BMW models such as i7 and X7 have the highest average prices in the dataset
• Petrol vehicles dominate the dataset, followed by diesel and hybrid vehicles
• Popular models like BMW 3 Series and 5 Series appear more frequently, indicating strong market demand
• Car prices show noticeable variation depending on model, engine size, production year, and mileage

🛠 Tools & Technologies Used

• Python
• Pandas
• Matplotlib
• Seaborn
• Jupyter Notebook

This project helped me further develop my skills in data cleaning, exploratory data analysis, and data visualization while working with a real-world style dataset. It also strengthened my ability to interpret data and communicate insights through visualizations.
